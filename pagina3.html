O que vamos ver?

Durante os estudos de Desenvolvimento Web 1, diversos conceitos e tecnologias foram aprendidos para a construção de aplicações modernas e eficientes, como exemplos:


HTML5

Utilizado para estruturar o conteúdo das páginas web, criando títulos, textos, imagens, tabelas, formulários e outros elementos.

CSS3

Responsável pela estilização das páginas, permitindo criar layouts modernos, cores, animações e efeitos visuais.

Bootstrap

Framework utilizado para facilitar a criação de interfaces responsivas que se adaptam a diferentes tamanhos de tela.

JavaScript

Linguagem utilizada para adicionar interatividade às páginas, tornando os sites mais dinâmicos e funcionais.

Responsividade

Técnicas utilizadas para garantir que o site funcione corretamente em computadores, tablets e smartphones.

Versionamento com Git

Ferramenta que auxilia no controle das alterações realizadas durante o desenvolvimento dos projetos.
